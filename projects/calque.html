Calque - Interactive Map Generator

A solution to create interactive maps for all purposes.

Project Description

Orienting oneself in space is a daily challenge. The experience becomes more complex when exploring an unfamiliar place. Maps remain a ubiquitous tool for overcoming this challenge, but the static nature of most maps has several flaws that affect their accessibility. This limitation hinders users' ability to navigate, plan routes, and obtain relevant information.
I am interested in providing an interactive map generation tool, with the idea of ​​making maps more accessible to users, but also to help map designers create them flexibly and maintain them.
Several interactive maps exist in different contexts. My goal is to provide an interactive map generation tool such that a map designer could easily provide an interactive map to his audience and export it in a format that is interesting for his institution. The designer could represent the points of interest, their properties, and draw the paths between the different points of interest to represent his network.
This project involves designing and developing a solution to this problem. The solution should allow users to build or load a map with annotations, simulate journeys (on the map), and suggest paths (routes) or destinations based on queries.

      • Edge: Represents connections between nodes.
      • Image: Handles image elements in the canvas.
    • Sidebars:

    • Navbar: Navigation bar for the application.
    • Footer: Footer component.
  3. State Management:
    • Redux: Used for global state management.
    • Contexts: Custom contexts for managing specific state (e.g., GraphContext, SelectedEntityContext).
  4. Styling:
    • Tailwind CSS: Used for styling components.
