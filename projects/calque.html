Tech Stack

  • Figma
  • Frontend: React.js
  • Styling: TailwindCSS
  • Backend: Node.js
  • API: Express.js
  • Database: MongoDB
  • Deployment: Netlify
  • Version Control: Git

Why This Tech Stack?

Figma for prototyping and UI/UX design, allowing for rapid iteration and feedback. React was chosen for its component-based architecture and strong community support. Tailwind to write CSS easily and quickly, allowing for a more efficient development process. Node.js for the backend, providing a non-blocking, event-driven architecture that is perfect for I/O-heavy applications. Express for its simplicity and flexibility, making it easy to set up a robust API. MongoDB for its NoSQL capabilities, allowing for flexible data models and easy scalability.
