The DND App

A character creating tool for Dungeons and Dragons

Project Description

This non-official, fanmade D&D app was created with the purpose of having an easy to use interface that automates most of the redundant stuff you need when creating a character. It focuses on what's important and auto-selects all the non-modifiable content like the class features or abilities. It lets you have your characters always with you, so you can access them even without your character sheet. Moreover, you can level them up easily, making you save time with all the search usually needed. Tech Stack

  • Figma
  • Frontend: React.js
  • Styling: SCSS + CSS
  • Backend: Supabase
  • Webscraping: Python

Why This Tech Stack?

Figma for prototyping and UI/UX design, allowing for rapid iteration and feedback. React was chosen for its component-based architecture and strong community support. Supabase was selected for its ease of use and integration with PostgreSQL, providing a robust backend solution without the need for extensive setup. SCSS was used for styling to take advantage of its nested rules and variables, making the CSS more maintainable.
