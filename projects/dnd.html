The DND App

A character creating tool for Dungeons and Dragons

Project Description

This non-official, fanmade D&D app was created with the purpose of having an easy to use interface that automates most of the redundant stuff you need when creating a character. It focuses on what's important and auto-selects all the non-modifiable content like the class features or abilities. It lets you have your characters always with you, so you can access them even without your character sheet. Moreover, you can level them up easily, making you save time with all the search usually needed. It also serves as a database for all the spells, classes, races, features, etc.

What I have done

  1. Character Creation System
    • Multi-step character creation process (7 steps)
    • Progress tracking with a progress bar
    • Ability to cancel creation with confirmation prompt
  2. Character Customization
    • Race selection with subraces
    • Class selection with subclasses
    • Background selection
    • Character identity customization (name, alignment, etc.)
  3. Stats System
    • Multiple stat generation methods:
      • Point Buy system
      • Custom system (with DM approval)
    • Ability to modify stats with bonus points
    • Stat tracking for all six core abilities (Strength, Dexterity, Constitution, Intelligence, Wisdom, Charisma)
  4. Feat System
    • Feat selection at specific level milestones (4, 8, 12, 16, 19)
    • Ability Score Improvement (ASI) option
    • Searchable feat list
    • Feat details display (description, bonuses, prerequisites)
  5. User Account Features
    • User authentication system
    • Character vault for saved characters
    • Ability to create multiple characters
    • User profile management
  6. Navigation and UI Features

    • Search functionality for various components
