Why This Tech Stack?

Figma for prototyping and UI/UX design, allowing for rapid iteration and feedback. React was chosen for its component-based architecture and strong community support. Bootstrap ensures rapid development of responsive UIs with minimal CSS overhead.

What I could improve.

Because this was a project done in 3 days, if given more time, I would add an actual backend to take care of the Authentification, save the messages sent between caregivers and residents, and add a database to store the messages sent between caregivers and residents.
